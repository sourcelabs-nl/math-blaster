# Resources: images, sounds, fonts, sprites, atlases, tile maps

KorGE loads assets through korlibs Virtual File Systems (Korio). Loads are `suspend` functions —
call them from `sceneInit`/`sceneMain` or other suspend contexts.

## Where assets live and how to reach them

- Put files in `src/commonMain/resources/`.
- **`resourcesVfs`** — global VFS mounted at that resources folder, available on every target:
  ```kotlin
  val bmp = resourcesVfs["gfx/player.png"].readBitmap()
  ```
- **`KR`** (KorGE 4.0.3+) — an autogenerated, type-safe accessor (like Android's `R`) for every
  file in the resources folder, with shortcuts for images and sounds and auto-built atlases for
  folders ending in `.atlas`. The Forge IDE shows previews for these.
  ```kotlin
  val slice = KR.gfx.player.read().slice()
  ```
- **`ResourcesRoot`** — a mountable VFS (accessible via the injector / Scene) with `resourcesVfs`
  mounted at `/`; lets you mount extra sources (cache folder, base URL).

## Bitmaps / images

```kotlin
val bmp: Bitmap = resourcesVfs["player.png"].readBitmap()
val slice: BmpSlice = resourcesVfs["player.png"].readBitmapSlice()
image(bmp) { anchor(Anchor.CENTER); smoothing = false }
```
All targets support at least PNG and JPEG. `smoothing = false` gives crisp pixel-art sampling.

## Sounds and music

```kotlin
val sound = resourcesVfs["sfx/jump.mp3"].readSound()   // fully decoded in memory (SFX)
val music = resourcesVfs["bgm/theme.mp3"].readMusic()  // streamed/decoded on the fly (long audio)

val channel = sound.play()
sound.play(3.times)
sound.play(infiniteTimes)

channel.volume = 0.5
channel.pitch  = 1.2          // OpenAL targets
channel.stop()
channel.await()               // suspend until finished
channel.await { current, total -> /* progress */ }
```
All targets support WAV and MP3; native/desktop also support OGG. **Web caveat:** browsers block
audio until the first user interaction (click/tap), so don't play sound before then.

## Fonts

Three kinds, all cross-target:

```kotlin
// Bitmap font (.fnt / XML):
val bmpFont = resourcesVfs["font.fnt"].readBitmapFont()
text("Hi", textSize = 32.0, font = bmpFont)

// Device font → bitmap font with auto atlas:
val deviceFont = BitmapFont("Arial", fontSize = 32)

// TTF font (render to a bitmap via Context2d):
val ttf = TtfFont(resourcesVfs["myfont.ttf"].readAll().openSync())
```

## Sprites & sprite animations

A `Sprite` plays a sequence of frames (a `SpriteAnimation`). Cheap to have many on screen when
they share one atlas.

```kotlin
// From a uniform sprite sheet grid:
val sheet = resourcesVfs["explosion.png"].readBitmap()
val explosionAnim = SpriteAnimation(
    spriteMap = sheet,
    spriteWidth = 128, spriteHeight = 128,
    marginTop = 0, marginLeft = 0,
    columns = 8, rows = 8,
    offsetBetweenColumns = 0, offsetBetweenRows = 0,
)
val explosion = sprite(explosionAnim)
explosion.playAnimationLooped()

// From an atlas, selecting frames by name prefix:
val atlas = resourcesVfs["adventurer.xml"].readAtlas()
val run = atlas.getSpriteAnimation(prefix = "run")
val player = sprite(run)
player.playAnimationLooped()
```

Sprite control:
```kotlin
sprite.spriteDisplayTime = 50.milliseconds      // per-frame time
sprite.playAnimation(spriteAnimation = run, times = 3, startFrame = 0, reversed = false)
sprite.playAnimationLooped(walk)
sprite.playAnimationForDuration(2.seconds, jump)
sprite.stopAnimation()
sprite.setFrame(0)
sprite.onAnimationCompleted.add { /* ... */ }   // also onAnimationStarted / onAnimationStopped
```

## Atlases

An `Atlas` packs many images into one texture (fewer GPU draw calls). Read with `readAtlas()`,
or let `KR` auto-generate one from a `*.atlas` resource folder. Use `atlas.getSpriteAnimation`
or grab individual slices for `image(...)`.

## Tile maps

For grid-based worlds. Three concepts: `TileSet` (the tile images), an `IStackedIntArray2`
(the grid of tile IDs, supports stacked layers per cell), and the `TileMap` view.

```kotlin
val grid = StackedIntArray2(width = 20, height = 15)
grid.push(0, 0, tileId)                          // stack a tile at a cell
grid.push(0, 0, TileInfo(tile = 5, flipX = true, rotate = false).data)

val map = tileMap(grid, tileset, repeatX = TileMapRepeat.NONE)

// When you mutate the grid after creation, wrap it so the cache re-renders:
map.lock { grid.push(3, 4, tileId) }
```

`TileMapRepeat` is `NONE` / `REPEAT` / `MIRROR`. `TileInfo` packs tile ID + offset + flip/rotate
flags into an Int. For very large sparse worlds use `SparseChunkedStackedIntArray2` with chunks.
Full editor integration (Tiled, LDtk) ships as separate KorGE Store modules
(`korge-tiled`, `korge-ldtk`).
